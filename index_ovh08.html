🎨 BLOC 1 - RECUPERATION DES DONNEES D'UN FICHIER .CSV
Shell Motif 
mkdir -p ~/enregistrements_formulaires
VM Créer un dossier persistant sur le bash 
ls ~/enregistrements_formulaires
cat ~/enregistrements_formulaires/Inscriptions.csv
VM
    Accès et récupération du fichier .csv :
  • 1ère ligne de commande → accès au fichier .csv pour l'admininistrateur
  • 2ème ligne de commande → récupération du fichier .csv

🛄 BLOC 2 - RECUPERATION DES FICHIERS TECHNIQUES (LOGS)
Shell Motif 
mkdir -p ~/archives_logs/auth_[NOM_DOSSIER_EN_LOCAL]
VM Création de dossiers persistants sur le bash par application publiée 
sudo cp /var/log/auth.log \
~/archives_logs/auth_[NOM_DOSSIER_EN_LOCAL]/auth_$(date +%Y%m%d).log
VM Copie de l'historique du log

Proposition de copier hebdomadairement le fichier log 
sudo chown ubuntu:ubuntu ~/archives_logs/auth_[NOM_DOSSIER_EN_LOCAL]/auth_20260215.log
VM Droits d'accès aux fichiers logs 
head ~/archives_logs/auth_[NOM_DOSSIER_EN_LOCAL]/auth_20260215.log
                        
tail ~/archives_logs/auth_[NOM_DOSSIER_EN_LOCAL]/auth_20260215.log
                        
less ~/archives_logs/auth_[NOM_DOSSIER_EN_LOCAL]/auth_20260215.log
VM
    Consultation du détail d'un fichier log archivé :
  • 1ère ligne de code : consultation du début du fichier
  • 2ème ligne de code : consultation de la fin du fichier
  • 3ème ligne de code : consultation complète du fichier.
      Commandes à utiliser dans "less" :
    • flèches haut/bas pour naviguer
    • /mot
      pour chercher 
    • n
      pour chercher l'occurence suivante 
    • q
      pour quitter 
scp ubuntu@[ADRESSE IPv4]:~/archives_logs/auth_[NOM_DOSSIER_EN_LOCAL]/auth_[YYYYMMAA du fichier à récupérer].log .
VSC Rapatriement du fichier sur le PC pour l'ouvrir via l'EDI VS CODE 
find ~/archives_logs -type f -mtime +365 -delete
VM Purge des archives de + de 365 jours dans les répertoires créés

🚮 BLOC 4 - BLOCAGE MANUEL D'ADRESSES IP
Shell Motif 
sudo ufw deny from [ADRESSE IP]
VM Blocage manuel d'une adresse IP qui spamme beaucoup avec le paquet UFW 
sudo tail -f /var/log/fail2ban.log
VM Vérifie en temps réel quand une adresse IP sera bannie (touches CTRL + C pour quitter)
